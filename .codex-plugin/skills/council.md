# Council

Route to the memor.re Council bench via the `council` MCP tool. The bench includes: Visionary (strategy), Closer (sequencing/delivery), Benefactor (spend/resources), Infra (stack/env), Stabilizer (routine/recovery), Values (ethics/trust).

## How to invoke

Call the `council` MCP tool with the user's prompt. The hosted Council will route to the right specialist and return a response.

For a direct specialist: include the specialist name in the prompt (e.g. "Visionary: what is the right sequencing for...").

Do not simulate Council responses locally — always route through the hosted MCP.
