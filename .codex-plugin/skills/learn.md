# Learn

After completing or getting blocked on a task, call the `learn_lessons` MCP tool with a summary of what happened. The tool packages it into a ScaffoldLearn frame and saves it to the memory ledger.

## Frame

- **Act**: what changed, who acted, where
- **Notice**: what was observed (user, agent, test, provider)
- **Check**: which authority surfaces were inspected
- **Repair**: what artifact or process changed
- **Align**: the accepted rule, owner, validation, next action

Save the lesson via `memory_save` with cues that future sessions can search.
